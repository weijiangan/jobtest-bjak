## Running
1. Refer to building instructions first
2. Run the following and you are ready to go
```sh
$ yarn start
```

## Building
1. Run `yarn`
2. Run `yarn build`

## Development
1. Run `yarn`
2. Run `yarn run start:dev`

## Testing
1. Run `yarn test`
